# Ultra-Powerful Web Scraper API

A **free, Node.js-based web scraper API** to fetch all content from any website.  
Supports static websites via **Axios + Cheerio**. (JS-heavy sites via Puppeteer only works on full servers like Render.)

---

## Features

- Scrape **title, meta description, headings (h1–h6)**  
- Fetch **all links and images**  
- Get **clean text content** (ads/scripts removed)  
- Word count, canonical URL, language, charset  
- OpenGraph (og:title, og:description, og:image)  
- Fast fallback: Axios + Cheerio for static sites  
- Optional: Puppeteer for JS-heavy sites (works on Render/Railway)
